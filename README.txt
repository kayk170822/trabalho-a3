Projeto Lista de Tarefas Copyright (c) 2024.

O projeto tem como objetivo criar uma lista de tarefas em Java, personalizável ao gosto do usuário.

Para o funcionamento da aplicação, é necessário ter o NetBeans instalado e o MySQL na versão 8.0.

O arquivo mysql-connector-j-8.4.0.jar serve para fazer a conexão entre o NetBeans e o MySQL. Junto temos o arquivo modelo_a3.mwb, que funciona somente no MySQL Workbench. Este arquivo é o diagrama do banco de dados utilizado no programa.

ATENÇÃO: Caso você, por algum motivo, não utilize o arquivo modelo_a3.mwb, é necessário criar um banco de dados do zero, com os parâmetros exatos mostrados nas fotos.
